おもいのほか時間はかかったがようやくリリースできました。

https://mikke.appがURLです。

// To AIここにapp-video.mp4を挿入してassets/videosの中にある。

以前のまとめた記事などはZennにあります。https://zenn.dev/hulk510/articles/my-first-service-mikke

今回は、webで作っていたものをブラッシュアップして色々頑張りアプリとしてリリースしました。

技術スタック
- react native
- expo
- typescript
- neon
- zitadel
- vercel

あたりがメインです。monorepoの構成で作成しており、型の共通化なども行っていたりします。

基本的にはreact nativeで作成してAPI箇所はwebの部分をある程度そのまま流用できたので比較的簡単に移行ができたかなーと思います。このタイミングでzitadelへの移行なども行なったりしていて色々と詰まる部分があったのでどこかで共有できたらなーと思います。

ただ、できていないかしょも。

実はzitadelでoauthをしているが、googleのoauthに関してはまだテスト環境でのリリースになっていました。そのせいでリンクできないとかもしばしば発生していて。なんか動いてはいるけど動いてないみたいな感じになっています。



ちょっと萌え切った感がある。
